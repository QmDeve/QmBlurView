## Characteristic
- **View**
  - `BlurView`
  - `BlurButtonView`
  - `ProgressiveBlurView`
  - `BlurTitlebarView`
  - `BlurSwitchButtonView`
  - `BlurFloatingButtonView`
- **Minimum support Android 5.0**
- **High Performance**: Native blur algorithm implemented with underlying `Native` calls
- **Automatic Recycling Mechanism**: Prevents memory leaks
