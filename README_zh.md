## 特性
- **View**
  - `BlurView` - 通用模糊视图
  - `BlurButtonView` - 模糊按钮视图
  - `ProgressiveBlurView` - 渐进模糊视图
  - `BlurTitlebarView` - 模糊标题栏视图
  - `BlurSwitchButtonView` - 模糊切换按钮视图
  - `BlurFloatingButtonView` - 模糊悬浮按钮视图
- **最低支持 Android 5.0**
- **高性能**：底层调用 `Native` 实现的原生模糊算法
- **自动回收机制**：防止内存泄漏
